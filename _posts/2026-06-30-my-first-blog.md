# 我的第一篇 Blog：使用 GitHub Pages 建立个人博客

这是我的第一篇 Blog 文章。

本次课程实践中，我使用 GitHub 创建了一个公开仓库，并把个人 Blog 的 Markdown 源文件保存到仓库中。随后，我配置了 GitHub Actions，使仓库内容在提交后能够自动发布到 GitHub Pages 网站。

通过这次实践，我初步了解了 GitHub 仓库的基本使用方法，也理解了 Markdown 文件可以通过静态网站生成工具转换成网页内容。同时，我还学习了 GitHub Actions 的自动化发布流程。

这次实践让我认识到，开源工具不仅可以用于代码管理，也可以用于个人知识整理、技术博客发布和项目展示。
